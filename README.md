🍽️ Reestar 

Reestar is a single-page restaurant menu card designed using HTML and CSS only. The project features a clean, modern, and visually appealing layout that showcases different food categories in an organized and user-friendly format.

The main objective of this project was to practice structuring web pages using semantic HTML and styling them with CSS without using any frameworks or JavaScript. It demonstrates proper layout design, spacing, alignment, and basic responsive design principles.

🔧 Technologies Used

HTML5

CSS3

This project helped strengthen my front-end development skills and understanding of webpage design fundamentals.
